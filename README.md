# Aphelion Systems Website

A static business website for a web, app, and digital systems studio. It is built with plain HTML, CSS, and JavaScript so it can run locally or deploy directly to GitHub Pages.

## Files

- `index.html` - concise homepage
- `services.html` - service and systems details
- `pricing.html` - suggested starting prices
- `about.html` - studio positioning and human presence
- `faq.html` - common client questions
- `process.html` - project process and standards
- `contact.html` - project inquiry form
- `styles.css` - responsive visual design
- `script.js` - mobile nav, theme toggle, and form status behavior
- `assets/hero-workspace.png` - generated hero artwork

## Preview

Open `index.html` in a browser, or serve this folder with any static file server.
